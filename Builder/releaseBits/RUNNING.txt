To run the bot, simply run 
java -jar ElizaBot-bin.jar

If you would like to change the parameters (or see the defaults), try with 
java -jar ElizaBot-bin.jar -h

Then simply add the user (by default socialblendeliza@jabber.org) to your 
XMPP client, and talk to it. Endlessly. :)
